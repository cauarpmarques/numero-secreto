# 🎯 Jogo do Número Secreto

## 📚 Projeto de Estudos

Este projeto foi desenvolvido como parte dos meus estudos em **HTML, CSS, JavaScript e Git/GitHub**, com o objetivo de praticar conceitos fundamentais do desenvolvimento web e o versionamento de código.

Durante a construção do projeto, explorei desde a estruturação de páginas web até a manipulação dinâmica de elementos utilizando JavaScript.

---

## 🚀 Objetivos de Aprendizagem

- Aprender a utilizar o Git e GitHub para versionamento
- Praticar HTML5 e CSS3
- Desenvolver lógica de programação com JavaScript
- Manipular elementos da página utilizando DOM
- Trabalhar com funções e estruturas condicionais
- Criar interfaces responsivas
- Organizar projetos Front-End

---

## 🎮 Sobre o Projeto

O jogo gera um número aleatório entre **1 e 50** e desafia o usuário a descobrir qual é o número secreto.

A cada tentativa, o sistema fornece dicas indicando se o número correto é maior ou menor que o valor informado.

Além disso, o projeto utiliza síntese de voz para tornar a experiência mais interativa.
